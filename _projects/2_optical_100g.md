## Overview

Developed a **100 Gbps (4 × 25 Gbps) NRZ SiGe BiCMOS optical transceiver IC** for medium-reach data center and telecom applications at Qualitas Semiconductor (2023–2024).

### Key Specifications

| Parameter | Value |
|-----------|-------|
| Data Rate | 100 Gbps aggregate (4 × 25 Gbps/lane) |
| Modulation | NRZ |
| Process | SiGe BiCMOS 180nm |
| Application | Medium-reach optical interconnect |

### Highlights

- **SiGe BiCMOS** process leveraging high-fT HBTs for wideband TIA and limiting amplifier
- Full transceiver: **TX modulator driver + RX TIA + CDR interface**
- Achieved target BER < 10⁻¹² at the specified reach
- Design methodology transferable to next-generation 400G PAM-4 architectures
