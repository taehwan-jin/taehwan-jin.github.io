## Overview

Developed an **IR-UWB (Impulse Radio Ultra-Wideband) CMOS distance sensor** as the core of my **Ph.D. dissertation** at Yonsei University (2008–2015). The system featured an adaptive time-of-arrival (ToA) measurement architecture with scalable resolution.

### Key Specifications

| Parameter | Value |
|-----------|-------|
| Technology | UWB impulse radio |
| Process | CMOS |
| Measurement | Time-of-Arrival (ToA) |
| Feature | Adaptive resolution scaling |
| Application | Indoor ranging and localization |

### Highlights

- Novel **adaptive ToA measurement** technique enabling scalable distance resolution
- **Highly linear LNA** designs using multiple gated transistors (MGTR) with capacitance desensitization and bulk-bias control techniques
- Frequency selective feedback LNA achieving **0.7 dB NF** with **+8.2 dBm IIP3**
- Published **4 first-author papers** in IEEE TMTT, IEEE MWCL, IEEE TCAS-I, and IJCTA
